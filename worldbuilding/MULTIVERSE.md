# The Multiverse — Dimensional Structure

## Three Scales

### Scale I: The Colosseum
The arena where regimes compete. Exists outside any single cosmology. The Cosmogonists built it — or discovered it (that question is itself a narrative thread). In RPG terms: the hub city. All characters return here between adventures.

**Contains:**
- The simulation engine (code runs here)
- The Chronicle archive (all Agon cycles recorded here)
- The Constitutional forge (consensus becomes law here)
- The Abbey Road intersection (where all symbolic streets converge)
- The Cosmogonists' observation platform (above the arena floor)

**Physics:** Neutral. No regime's rules dominate. The only physics are the meta-laws (Section 7 of design spec) and the Colosseum's own rules of engagement.

### Scale II: The Regime Realms
Each regime controls a realm — a complete world built according to its philosophy. When a regime takes the reins of the Colosseum, its realm EXPANDS — its aesthetic, rules, and characters bleed into the shared space. When it yields, its realm contracts back.

| Realm | Cosmology | Environmental Logic |
|-------|-----------|-------------------|
| **Arda Perpetua** | Tolkien | Delegated stewardship. Beauty preserved through constraint. Environmental responsiveness — the land reflects moral state. |
| **The Pressure Fields** | Malazan | Landscape shaped by accumulated historical force. Warrens open and close. Ascendancy thresholds visible as atmospheric distortions. |
| **The Construct** | Matrix | Rendered reality. Green-tinted. Architecture that can be loaded and unloaded. Programs as NPCs. Glitches as features. |
| **The Lodge Territories** | Lynch | Geography shifts when unobserved. Multiple locations occupy the same coordinates. Electricity hums beneath surfaces. |
| **Hyrule Eternal** | Zelda | Triadic geography — three regions corresponding to Power, Wisdom, Courage. Temples as governance nodes. The cycle visible in seasonal shifts. |
| **The Pattern Ground** | WoT | Woven landscape — thread-like structures visible in the sky. The Taint appears as dark staining spreading from one direction. |
| **The Grand Line** | One Piece | Maritime world — islands as truth-fragments. Each island holds a piece of the Poneglyph. Navigation requires reconstructing the full picture. |
| **The Island** | Lost | Jungle-covered landmass that moves. Time is non-linear. The Source glows beneath. Numbers recur in everything. |
| **The Pose Dimension** | JoJo | Reality saturated with style. Colors more vivid. Sound effects visible. Names carry power. Music is a martial art. |

### Scale III: The Folds
Within each realm, significant decisions create folds — branching timelines. Folds are the multiverse's primary expansion mechanism.

**Fork creation:** Any significant event in any realm (a regime audit finding, a character choice, a community vote) can create a fork. The fork generates two or more paths. Each path becomes its own fold — a navigable, explorable alternate timeline.

**Fold physics:**
- Folds inherit the parent realm's physics but may drift
- The longer a fold exists, the more it diverges from the parent
- Folds can be revisited but not retroactively edited (the Witnesses ensure this)
- When folds produce data relevant to the Colosseum, it feeds up to Scale I

**The multiverse tree:** Folds accumulate into a branching tree structure. The tree is:
- Navigable (characters can walk between folds through liminal rooms)
- Visualizable (Three.js render of the branching structure — ORGAN-II art)
- Analyzable (divergence metrics computed for each fork — META data)
- Publishable (the tree itself is a creative/commercial artifact — ORGAN-III)

## Dimensional Navigation

Characters move between scales through specific mechanisms:

| From → To | Mechanism | Who Can Do It |
|-----------|-----------|--------------|
| Colosseum → Realm | Regime summon portal | Any character during active Agon |
| Realm → Colosseum | Return gate (appears when Agon cycle ends) | Any character |
| Realm → Fold | Fork decision point | Any character at the fork |
| Fold → Parent Realm | Fold collapse (voluntary exit) | Any character |
| Realm → Realm | Symbolic street corridor | Psychopomps and characters with corridor access |
| Any → Liminal room | Emotional trigger or recursive instability | Involuntary or Cosmogonist-granted |
| Liminal → Primary room | Stabilization (anchoring to H.E.R.E.) | Any character with anchor |

## The Phaethon Mechanic at Scale

When a regime reigns and its phaethon blind spot manifests:

1. The regime's realm begins to **bleed** into the Colosseum (visual: the Colosseum takes on the regime's aesthetic involuntarily)
2. Other realms' borders weaken (characters from those realms report "glitches" — Lynchian intrusions)
3. The Cosmogonists issue a Phaethon Warning
4. If uncorrected: the corrective regime is forcibly summoned (the Colosseum self-heals)
5. The event is recorded as a **Phaethon Chronicle entry** — the mythological equivalent of "someone almost burned the world"

This is the chariot of the sun. Phaethon asked to drive. The earth scorched. The system remembers.
