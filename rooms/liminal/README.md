# Liminal Rooms

Individual room specifications for the four liminal spaces.
Defined in ROOMS.md — this directory will contain expanded individual files.

- NOWHERE (`_`) — the null room, void, recursion base case
- SOMEWHERE (`~`) — ambiguous memory, unresolved recursion
- BACKTHEN (`<<`) — false past, nostalgia glitch
- NEVERW4S (`?!`) — counterfactual timelines, rejected folds
