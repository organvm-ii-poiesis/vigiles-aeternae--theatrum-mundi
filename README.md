# Vigiles Aeternae — Theatrum Mundi

**The Theater of the World** — a creative multiverse, RPG system, worldbuilding bible, and spatial navigation architecture for the Vigiles Aeternae franchise.

## What This Is

The Theater where the Agon is performed. Watchers take physical form here. The multiverse branches here. Every forking thread is a stage. Characters, environments, powers, fusions, magic systems, and recursive rooms — all alive, all playable, all evolving.

## Structure

```
worldbuilding/      Multiverse skeleton, realm definitions
characters/
  orders/           Character sheets per Watcher Order
  regimes/          Regime-specific characters
rpg/                RPG system rules, magic system, classes
rooms/
  primary/          H.E.R.E., T.H.E.R.E., E.V.E.R.Y.W.H.E.R.E., M41N_STR33T
  liminal/          NOWHERE, SOMEWHERE, BACKTHEN, NEVERW4S
  streets/          Symbolic street corridors
art/                Generative art code, visual assets
sound/              Sonic identities, regime soundscapes
narrative/          Forking threads, chronicles, fold records
bestiary/           Mythological beings as interactive entities
```

## The Magic System

| Mechanic | Function |
|----------|----------|
| **Absorb** | Learn a power by studying it |
| **Fuse** | Combine two powers into a hybrid |
| **Evolve** | Refine through Agon cycles |
| **Replicate** | Spawn adversarial copies |
| **Relinquish** | Wear another's stats temporarily |
